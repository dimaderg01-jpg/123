Из чего состоит доменное имя сайта: разбираем иерархию

Пример: dimaderg01-jpg.github.io

Давайте разложим этот адрес по уровням:

  • Корневой домен (root) — невидимая точка в самом конце: dimaderg01-jpg.github.io.
  • Домен верхнего уровня (TLD) — .io
    Это национальный домен Британской территории в Индийском океане, но сейчас популярен среди IT-проектов.
  • Домен второго уровня (SLD) — .github
    Здесь это имя сервиса GitHub Pages, который хостит мой сайт.
  • Домен третьего уровня (субдомен) — dimaderg01-jpg
    Это мое уникальное имя пользователя в системе GitHub. Весь домен dimaderg01-jpg.github.io — это мой личный сайт на GitHub Pages.
